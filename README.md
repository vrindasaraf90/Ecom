# Ecom
I have create an E-commerce app for Customer and Shopkeeper both.
For the project details watch the tutorial https://youtu.be/jQWKaGtZCqM ; https://youtu.be/x_zixlLEPEY.


Features of our Application:


CUSTOMER
- ADD product to cart
- EDIT product 
- DELETE product
- PLACE AN ORDER


SHOPKEEPER
- ADD product
- EDIT product
- DELETE product
